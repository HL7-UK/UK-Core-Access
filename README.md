# UK-Core-Access

## Build
### Prerequisites
NodeJS

### Install
Clone repository from github

Import dependencies
```
$ npm i
```

### Build
```
$ sushi ./ig/UKCoreAccess
```

## Publish
### Prerequisites
Java Runtime 55+

### Install
```
$ cd ./ig/UKCoreAccess
$ ./_updatePublisher.sh
```
### Build
```
$ ./genonce.sh
```
